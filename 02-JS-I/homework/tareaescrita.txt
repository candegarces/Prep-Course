Variables: una variable es una forma de guardar el valor de algo para usarlo mas tarde. Se utiliza la palabra clave "var" seguida de un espacio y el nombre de la variable
Strings: es cualquier texto que se encuentre encerrado entre comillas ya sean simples '' o complejas ""
Funciones: son tipos de objetos invocables que se encargan de cambiar, calcular o generar algo con las variables. Una función debera tener paréntesis en los cuales ira una variable o multiples argumentos y corchetes donde ira el codigo que ejecutaremos. En las funciones solemos utilizar la declaracion "return" en esta nosotros obtendremos la devolucion de una variable de cualquier tipo, por ejemplo el resultado de una ecuación matemática.
Declaración if: en este tipo de declaración compararemos dos o mas tipo de argumentos en los cuales deberemos dar una respuesta si cumplen o no cumplen con lo especificado. En este caso utilizaremos operadores como: <, >, <=, >=, ===, !==.
Valores booleanos: este concepto significa que hay dos opciones activar o desactivar, que serian true o false (si algo es verdadero o falso). Este dato se relaciona con la definicion de declaración if ya que a partir de estos podremos confirmar si lo que se esta condicionando es verdadero o falso.
